# Expected Outcomes

Expected separation results:

1. Alert delivery can remain high while useful-event fraction drops.
2. Delayed telemetry can arrive but become stale for triage.
3. Duplicated or flooded alerts can increase volume without improving belief.
4. Adversarial suppression can create false confidence.
5. Uncertainty-aware or usefulness-aware triage can outperform severity-first triage under impairment.
