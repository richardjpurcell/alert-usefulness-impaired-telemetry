# Concept Note

## Working title

Alert Usefulness Under Impaired Security Telemetry

## Core claim

Security telemetry that arrives is not always useful.

## Research question

Can alert delivery, alert volume, or apparent telemetry coverage remain high
while defender knowledge about incident state fails to improve?

## Intended contribution

A reproducible audit framework for distinguishing delivered security telemetry
from useful security evidence under operational and adversarial impairment.
