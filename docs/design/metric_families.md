# Metric Families

| Family | Example metric | Interpretation |
|---|---|---|
| Delivery | alert/event delivery rate | What arrived |
| Timing | stale-event fraction | Whether it arrived in time |
| Usefulness | useful-event fraction | Whether it improved defender knowledge |
| Belief quality | belief entropy, belief error | What the defender knows or misunderstands |
| Misleadingness | false-confidence rate | Whether telemetry made belief worse |
| Effort | entropy reduction per analyst action | Cost of useful evidence |
