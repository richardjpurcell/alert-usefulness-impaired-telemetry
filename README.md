# ASTRA: Alert/Telemetry Security Research Audit Tool

ASTRA is a bounded diagnostic research instrument for studying whether delivered
security telemetry is useful for maintaining a defender's estimate of incident
state under delay, loss, noise, duplication, overload, and adversarial impairment.

Core claim:

> Security telemetry that arrives is not always useful.

This project is intended to support a research paper on alert usefulness auditing
for SOC/SIEM-style telemetry pipelines.
